== Description ==

Screen Reader adds a floating control panel to your site, allowing users to listen to page content or read text on hover. All controls are generated by JavaScript, no HTML output from PHP.

== Installation ==
1. Upload the `screen-reader-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Does it work in all browsers? =
Supports Chrome, Firefox, Safari, and Edge (speechSynthesis API required).

= Can I customize the control panel? =
You can override styles in your theme or edit the JS/CSS files.
